## 🖥️ Lo que hago en mi tiempo

### Self-hosting & Servidores

Mantengo un servidor propio con servicios autoalojados:

- **[Immich](https://immich.app/)** — Alternativa self-hosted a Google Photos para gestión de fotos
- **[OpenWebUI](https://openwebui.com/) + [Ollama](https://ollama.com/)** — Interfaz web para modelos de IA corriendo localmente, sin depender de APIs externas
- **Pipelines personalizados** — Scripts en Python que extienden las capacidades de OpenWebUI (filtros, funciones, integraciones)
- **[Tailscale](https://tailscale.com/)** — VPN para acceso seguro remoto a todos los servicios

## 📝 Filosofía de aprendizaje

No me conformo con que "funcione". Necesito entender **por qué** funciona.

Si instalo Docker, quiero saber cómo levanta un contenedor. Si uso React, quiero entender qué pasa con el Virtual DOM. Si corro un modelo de IA local, quiero saber cómo se conecta todo.

Esa obsesión por entender las capas inferiores es lo que me diferencia.
